Setup Instructions
How to add a ZKTeco biometric device:
  1. Connect the device to your network using an Ethernet cable
  2. Find the device's IP address from the device menu: Network → IP Address
  3. Ensure the device is accessible from your network (try pinging the IP)
  4. Enter the device name, IP address, and port above (default port is 4370)
  5. Click "Add Device" to save the configuration
  6. Go to the device detail page and click "Test Connection" to verify connectivity
  7. Click "Sync Users" to import enrolled users from the device
Note: Make sure the device's IP address is static and doesn't change via DHCP.
