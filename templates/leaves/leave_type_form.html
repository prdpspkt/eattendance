Understanding Leave Type Settings:
  • Name & Code: Display name and short code for reports (e.g., Annual Leave - AL)
  • Days Per Year: Maximum number of days allowed per year for this leave type
  • Is Paid: Whether employees receive salary during this leave
  • Requires Approval: Whether leave requests need manager approval before taking effect
  • Active: Only active leave types are available for selection in leave requests
Example: Annual Leave with 20 days/year, paid, requires approval
- Employees can request up to 20 days per year
- They receive full salary during leave
- Manager must approve before leave is granted
